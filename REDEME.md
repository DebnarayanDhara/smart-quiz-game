# Python Quiz Game

A simple and fun Quiz Game built with Python.  
This game asks 10 random questions from a large question bank and calculates your score, percentage, and grade.

---

## Features

- Randomly selects 10 questions
- Questions from:
  - Mathematics
  - Science
  - General Knowledge
  - History
  - Computer
  - English Grammar
- Case-insensitive answer checking
- Calculates:
  - Score
  - Percentage
  - Grade
- Option to play again
## How to Run
1. Install Python
2. Run This Program

## How to Play
1. The game will ask 10 random questions.
2. Type your answer and press Enter.
3. The program will check whether your answer is correct.
4. At the end, you will see:
     Total Score
     Percentage
     Grade
5. You can choose to play again.

## Grade System
| Percentage |   Grade   |
| ---------: | :-------: |
| 90% - 100% | Excellent |
|  80% - 89% |     A     |
|  70% - 79% |     B     |
|  60% - 69% |     C     |
|  50% - 59% |     D     |
|  Below 50% |     F     |
